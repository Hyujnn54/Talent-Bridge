TROUBLESHOOTING:

Problem: No scheduler startup message
  → Check MainFX calls InterviewReminderScheduler.start()
  → Check for errors before the banner

Problem: "Total interviews in DB: 0"
  → Check database has interview records
  → Check interview table connection is working

Problem: "SEND_NOW" but no email/SMS messages
  → Check Brevo API key in email.properties
  → Check SMS API key in sms.properties
  → Check candidate email/phone in users table

Problem: Same reminder sends twice
  → Check reminder_sent column was added
  → Check column is being set to TRUE
  → Verify database update message appears

Problem: Interview marked as "TOO_LATE" too early
  → Check system time is correct
  → Check interview scheduled_at is in future
  → Check DEBUG_MODE is TRUE for testing
